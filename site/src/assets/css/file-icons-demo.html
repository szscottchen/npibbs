🔧 使用方法

HTML 使用示例：

<span class="file-link-wrapper"> <i class="file-icon file-icon-excel"></i> <a href="/uploads/files/document.xlsx">财务报表.xlsx</a> </span>

Vue.js 使用示例：

<span class="file-link-wrapper"> <i class="file-icon" :class="getFileIconClass(file)"></i> <a :href="file.url">{{ file.name }}</a> </span>

JavaScript 函数调用：

// 获取文件图标类名 const iconClass = getFileIconClass(fileObject); // 返回示例："file-icon file-icon-word"
